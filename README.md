# Personal Portfolio Website

A responsive **Personal Portfolio Website** built using **HTML**, **CSS**, and **JavaScript**. This project showcases personal information, skills, projects, and contact details with a clean, modern, and mobile-friendly design.

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## ⚙️ Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/A-P-Singh1/Personal-Portfolio-Website.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Personal-Portfolio-Website
   ```

3. Open the `index.html` file in your web browser.

   **OR**

   Use **Visual Studio Code** with the **Live Server** extension:
   - Open the project folder.
   - Right-click `index.html`.
   - Select **Open with Live Server**.
